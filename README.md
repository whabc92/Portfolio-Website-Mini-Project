A modern portfolio website built with **React** and **Vite**, showcasing projects, skills, and contact information.

---

## Features

- Fully responsive design for desktop, tablet, and mobile  
- Smooth animations and modern theme  
- Contact form integrated with **Web3Forms**  
- Easy to customize and extend  
- Environment variables used for public API keys  

---

## Tech Stack

- **React** – Frontend library for building UI  
- **Vite** – Fast development server and build tool  
- **CSS** – Custom styling  
- **Web3Forms** – Handling contact form submissions
